## 🛡️ Giải pháp kiến trúc thay thế

Thay vì patch từng caller, có thể đặt **default `per_model_timeout`** bên trong `trinity_bridge.run()`:

```python
# Nếu caller không truyền per_model_timeout, tự tính dựa trên global timeout và số model
if pmt is None:
    pmt = min(t / max(len(models), 1), 30.0)  # Mỗi model tối đa 30s hoặc chia đều
```

> [!IMPORTANT]
> Giải pháp này áp dụng toàn cục, bảo vệ TẤT CẢ callers mà không cần sửa từng file. Tuy nhiên cần cẩn thận với các task nặng (neural_rewriter, plagiarism check) có thể cần thời gian generation dài hợp lệ.
